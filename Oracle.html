JOBS OFFERED BY

Oracle offers various roles for B.Tech Computer Science Engineering (CSE) graduates, providing opportunities in software development, data science, cloud computing, and security domains. Below are some of the key roles:

  1. Software Engineer
    • Role Overview:
      Software Engineers at Oracle work on designing, developing, and maintaining scalable software applications. They collaborate with product managers, UX designers, and other engineers to build robust enterprise solutions. Responsibilities include writing clean and efficient code, debugging, and optimizing application performance.
    • Skills Required:

      ->Proficiency in Java, Python, or C++
      ->Strong understanding of Data Structures & Algorithms
      ->Experience with Cloud Computing (Oracle Cloud, AWS, Azure)
    • Salary Range:
      ₹10 LPA - ₹25 LPA
  2. Machine Learning Engineer
    • Role Overview:
      Machine Learning Engineers at Oracle design, develop, and deploy ML models for predictive analytics and automation. They work with large datasets, train models, and optimize algorithms to enhance business intelligence.
    • Skills Required:

      ->Proficiency in Python, R, or Scala
      ->Experience with TensorFlow, PyTorch, Scikit-learn
      ->Strong understanding of Mathematical Concepts & Probability
    • Salary Range:
      ₹12 LPA - ₹30 LPA
  3. Data Scientist
    • Role Overview:
      Data Scientists analyze complex datasets to extract insights that drive decision-making. They develop data models, perform statistical analysis, and visualize data trends.
    • Skills Required:

      ->Expertise in Python, R, SQL
      ->Experience with Machine Learning & Deep Learning algorithms
      ->Strong understanding of Data Wrangling & Feature Engineering
    • Salary Range:
      ₹12 LPA - ₹28 LPA
  4. Cloud Engineer
    • Role Overview:
      Cloud Engineers at Oracle develop, deploy, and manage cloud-based solutions for enterprise applications. They ensure cloud security, scalability, and cost optimization.
    • Skills Required:

      ->Experience with Oracle Cloud, AWS, or Microsoft Azure
      ->Knowledge of Containerization & Orchestration (Docker, Kubernetes)
      ->Understanding of Infrastructure as Code (Terraform, Ansible)
    • Salary Range:
      ₹15 LPA - ₹32 LPA
  5. DevOps Engineer
    • Role Overview:
      DevOps Engineers automate and optimize software development, testing, and deployment processes. They work on integrating CI/CD pipelines and managing cloud infrastructure.
    • Skills Required:

      ->Experience with CI/CD Tools (Jenkins, GitHub Actions)
      ->Proficiency in Scripting Languages (Python, Bash)
      ->Knowledge of Infrastructure as Code (Terraform, Ansible)
    • Salary Range:
      ₹14 LPA - ₹29 LPA
  6. Security Engineer
    • Role Overview:
      Security Engineers at Oracle protect enterprise systems from cyber threats. They work on vulnerability assessments, encryption, and network security.
    • Skills Required:

      ->Knowledge of Cybersecurity & Ethical Hacking
      ->Experience with Security Tools (SIEM, IDS/IPS)
      ->Understanding of Encryption & Cryptography
    • Salary Range:
      ₹13 LPA - ₹27 LPA
  7. Database Administrator
    • Role Overview:
      Database Administrators (DBAs) manage and optimize Oracle databases, ensuring data integrity, security, and performance. They work on backup and recovery solutions.
    • Skills Required:

      ->Expertise in Oracle Database, MySQL, or PostgreSQL
      ->Knowledge of SQL Query Optimization
      ->Experience with Database Performance Tuning
    • Salary Range:
      ₹10 LPA - ₹25 LPA
  8. Product Manager
    • Role Overview:
      Product Managers at Oracle define the vision, strategy, and roadmap for products. They collaborate with engineers, designers, and business teams to bring products to market.
    • Skills Required:

      ->Strong understanding of Agile & Scrum Methodologies
      ->Ability to conduct Market Research & Competitive Analysis
      ->Experience with Product Lifecycle Management
    • Salary Range:
      ₹15 LPA - ₹35 LPA
  9. Quality Assurance Engineer
    • Role Overview:
      QA Engineers develop and execute test plans to ensure software quality. They focus on automation, performance testing, and bug tracking.
    • Skills Required:

      ->Experience with Automation Testing Tools (Selenium, JUnit)
      ->Knowledge of Manual Testing & Test Case Design
      ->Proficiency in Performance Testing (JMeter, LoadRunner)
    • Salary Range:
      ₹10 LPA - ₹22 LPA
