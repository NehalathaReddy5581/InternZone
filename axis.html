JOBS OFFERED BY

Axis Bank offers various roles for Computer Science Engineering (CSE) graduates, primarily in its IT, digital banking, and data analytics divisions. Below are some key job roles:

  1. Software Engineer / Developer :

    • Role:
      Develops and maintains banking applications, ensures software efficiency, and troubleshoots technical issues.
    • Skills Required:
      Java, Python, SQL, cloud computing, and API development.
    • Salary Range:
      ₹5,00,000 - ₹12,00,000 per annum.
  2. Data Scientist / Data Analyst :

    • Role:
      Analyzes customer and transactional data to provide insights for decision-making and fraud detection.
    • Skills Required:
      Machine learning, AI, data visualization, Python, R, SQL
    • Salary Range:
      ₹8,00,000 - ₹18,00,000 per annum.
  3. Cyber Security Analyst
    • Role:
      Ensures data security, manages cybersecurity risks, and prevents hacking threats.
    • Skills Required:
      Ethical hacking, penetration testing, cryptography, firewall management
    • Salary Range:
      ₹6,00,000 - ₹15,00,000 per annum.
  4. IT Support Engineer
    • Role:
      Manages IT infrastructure, supports banking operations, and troubleshoots network issues.
    • Skills Required:
      Network administration, cloud computing, system troubleshooting.
    • Salary Range:
      ₹4,50,000 - ₹10,00,000 per annum.
  5. AI & Automation Engineer
    • Role:
      Implements AI-driven chatbots, automation in banking processes, and predictive analysis.
    • Skills Required:
      Artificial intelligence, NLP, RPA, TensorFlow.
    • Salary Range:
      ₹7,00,000 - ₹16,00,000 per annum.
  6. Blockchain Developer
    • Role:
      Develops secure blockchain solutions for banking transactions and smart contracts.
    • Skills Required:
      Solidity, Ethereum, Hyperledger, cryptographic security.
    • Salary Range:
      ₹9,00,000 - ₹20,00,000 per annum.
  7. Cloud Engineer
    • Role:
      Manages cloud-based banking applications and ensures smooth cloud transitions.
    • Skills Required:
      AWS, Azure, Google Cloud, Kubernetes, Docker.
    • Salary Range:
      ₹7,50,000 - ₹14,00,000 per annum.
  8. Technical Business Analyst
    • Role:
      Bridges the gap between technical teams and business operations to optimize banking solutions.
    • Skills Required:
      SQL, Python, Excel, data visualization, project management.
    • Salary Range:
      ₹6,00,000 - ₹12,00,000 per annum.
  9. DevOps Engineer
    • Role:
      Automates banking software deployment, manages CI/CD pipelines.
    • Skills Required:
      Jenkins, Docker, Kubernetes, Ansible, Linux scripting.
    • Salary Range:
      ₹7,00,000 - ₹14,00,000 per annum.
  10. UI/UX Designer
    • Role:
      Designs user-friendly banking applications and websites.
    • Skills Required:
      Figma, Adobe XD, CSS, JavaScript, React.
    • Salary Range:
      ₹5,00,000 - ₹11,00,000 per annum.

Job Locations
India:
Primarily in Mumbai, Bangalore, Pune, Hyderabad, and Chennai.
Onsite (International):
Axis Bank collaborates with international fintech firms, and some tech professionals may get opportunities abroad.
